Overview:-
This is an Android Image Classification Application using TensorFlow Lite to classify images into predefined categories. Users can either take a picture via the camera or select an image from the gallery. The app predicts the class of the image and displays the result instantly.
This project is developed in Android Studio with Java, using ConstraintLayout for UI.

Features:-
Capture an image using the device's camera.
Select an image from the gallery.
Real-time image classification using the MobileNetV1 0.1 224 Quantized model.
Displays prediction result immediately.
Clean and responsive UI with a splash screen.

Automatically clear previous predictions when an image is either selected or captured.
Tech Stack:
Language: Java
IDE: Android Studio
Machine Learning Framework: TensorFlow Lite
UI: ConstraintLayout

Model: MobileNetV1 0.1 224 Quantized
Android SDK: Minimum SDK 21+
Usage:-
1. Open the app: the splash screen will appear
2. Click on either Capture Image or Select Image to input an image.
3. The app will output the predicted class.
4. The selection of a new image or taking a new photo automatically clears the previous result.
